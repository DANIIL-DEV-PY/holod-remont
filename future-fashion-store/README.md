# NOVA THREAD

Самостоятельный статический сайт магазина одежды в футуристичном стиле.

## Что внутри

- `index.html` - разметка витрины, каталога, корзины и оформления заказа.
- `styles.css` - футуристичная тема, адаптив, каталог и подборки образов.
- `script.js` - фильтры по размеру, цвету и категории, корзина, оформление, preview.
- `assets/hero-runway.png` - локальный fashion-баннер.
- `assets/products/` - локальные изображения товаров для карточек и корзины.

## Как открыть

Откройте `index.html` в браузере. Сборка и интернет не нужны.
