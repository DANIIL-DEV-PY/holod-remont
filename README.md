# Холод Сервис — сайт мастера по ремонту холодильного оборудования и бытовой техники

Одностраничный сайт-визитка на чистом HTML/CSS/JavaScript, без сборщиков и зависимостей.

## Возможности

- Список услуг с ценами; на телефоне карточки «наезжают» друг на друга при скролле (sticky-стек), на десктопе — сетка
- Отдельная страница для каждой услуги («Узнать больше»)
- Плавающая кнопка и модальное окно «Связаться с нами» (телефон + ВКонтакте)
- Адаптивная вёрстка под телефон (mobile-first)

## Как запустить локально

Просто откройте `index.html` в браузере — всё работает без сервера.

## Как заменить контакты и цены

- Телефон и ссылка ВКонтакте — в начале файла [`js/components.js`](js/components.js)
- Список услуг, цены, короткие описания — в [`js/services-data.js`](js/services-data.js)
- Полные тексты услуг — в HTML-файлах папки [`services/`](services/)

## Структура

```
index.html            — главная (hero, лента услуг, о мастере)
services/*.html        — страницы отдельных услуг
css/styles.css         — стили и цветовая палитра
js/services-data.js    — данные об услугах (единый источник)
js/components.js        — общий хедер/футер + контакты
js/script.js           — рендер списка услуг и логика модалки
js/track.js            — счётчик посещений (см. «Админ-панель» ниже)
admin/index.html       — админ-панель со статистикой посещений
netlify/functions/     — серверные функции: учёт визитов, вход в админку, отдача статистики
```

## Админ-панель и статистика посещений

На сайте есть защищённая паролем страница `/admin` со статистикой: просмотры и уникальные
посетители по дням за последние 30 дней, суммы за текущую и прошлую неделю, простой график.

Данные считаются через Netlify Functions и хранятся в Netlify Blobs (встроенное хранилище
Netlify — отдельная база данных не нужна).

### Настройка (один раз, после деплоя на Netlify)

1. В панели Netlify откройте ваш сайт → **Site configuration → Environment variables**.
2. Добавьте две переменные:
   - `ADMIN_PASSWORD` — пароль, под которым вы будете заходить в `/admin` (придумайте свой).
   - `SESSION_SECRET` — случайная строка для подписи сессии, например:
     `6d4fc4fc35c9436959f4aea97a57ae27c4db71cef29be647cba03f98170f7495`
     (можно использовать это значение или сгенерировать своё — главное, никому его не показывать).
3. Сделайте redeploy сайта (или просто запушьте изменения — Netlify пересоберёт сайт автоматически).
4. Откройте `https://ваш-сайт.netlify.app/admin`, введите пароль из `ADMIN_PASSWORD`.

Просмотры автоматически считаются на всех страницах сайта — ничего дополнительно включать не нужно.
